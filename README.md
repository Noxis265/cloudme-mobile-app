# CloudMe — обновление: чаты и видеозвонки

## Что нового
- Общий чат, который видят все
- Личные сообщения 1-на-1 (кнопка 💬 на карточке друга в разделе «Сеть»)
- Видеозвонки между друзьями (кнопка 📞 внутри личного чата)

Файлы и видео-вложения пока не добавлены — это бы требовало привязки карты к Firebase (Cloud Storage больше не работает на бесплатном тарифе без неё). Если решишь, что готов — скажи, добавим отдельно.

## Что нужно обновить в Firebase Console

Открой свой проект на https://console.firebase.google.com → раздел **Databases & Storage → Firestore → Rules** → полностью замени текст на содержимое файла `firestore.rules` из этой папки → **Publish**.

Больше никаких новых сервисов включать не нужно — всё работает на тех же Authentication и Firestore, что уже настроены.

## Пересобираем .exe

Никаких новых npm-пакетов не требуется, поэтому `npm install` заново запускать не нужно. Открой терминал в папке проекта и выполни:

```
npm run build
```

Новый `CloudMe Setup.exe` появится в папке `dist`. Установи его (он заменит старую версию) и отправь обновлённый файл друзьям — им нужно тоже переустановить.

## Камера и микрофон для звонков

При первом звонке Windows может спросить разрешение на доступ к камере/микрофону — нужно разрешить. Если звонок не подключается совсем (ни видео, ни звука), проверь в Параметры → Конфиденциальность и безопасность → Камера / Микрофон, что включено «Разрешить классическим приложениям доступ к камере/микрофону».

## Важно про надёжность звонков

Звонки идут напрямую между компьютерами друзей (без сервера-посредника), поэтому иногда — особенно если кто-то сидит за мобильным интернетом или сложным роутером — соединение может не установиться. Это нормально для бесплатного решения; если будет часто не работать, можно потом подключить платный сервер-релей (TURN), но это отдельная история.

## Готовый ярлык с иконкой (без electron-builder)

В папке теперь есть `CloudMe-icon.ico`, `Install.ps1` и `Установить CloudMe.bat`. Это самодостаточное решение: при первом запуске `Установить CloudMe.bat` сам создаёт на рабочем столе ярлык **CloudMe** с иконкой облака — с правильным путём именно туда, куда распакован архив — и сразу открывает приложение. Дальше пользуешься только этим ярлыком на столе, про .bat можно забыть.

**Как собрать архив для друга:**
1. У себя в папке проекта убедись, что `npm install` уже выполнен (папка `node_modules` на месте).
2. Заархивируй всю папку целиком — включая `node_modules`, `CloudMe-icon.ico`, `Install.ps1` и `Установить CloudMe.bat`. Архив будет большим (150–300 МБ) — отправляй через Telegram или ссылку на Яндекс.Диск/Google Drive.

**Что делает друг:**
1. Распаковывает архив в любую папку.
2. Один раз запускает `Установить CloudMe.bat`.
3. На рабочем столе появляется иконка CloudMe, приложение открывается. Дальше друг просто кликает по этой иконке, как по обычной программе — никаких .bat и терминалов больше не нужно.
